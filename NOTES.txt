1) Creating a new user:

	Create <username>        -- Cannot contain periods
	Create Group <username>
	Add 'inkspot' user to the group
	
	Create Directories:

		/home/users/<username>
		/home/users/<username>/mail/local
		/home/users/<username>/mail/<domain>
		/home/users/<username>/www/local/<user>/docroot
		/home/users/<username>/www/<domain>/docroot

	- SFTP Only access is enabled, public keys required
	- Adding public keys adds to /home/username/.ssh/authorized_keys
	- Trusting a user adds them to your group (can edit ~/www)

2) Creating a new domain creates a group and directory

	/home/domains/<domain>
	/home/domains/<domain>/www/docroot
	/home/domains/<domain>/mail
	/home/domains/<domain>/mail/<user>

	- Creator is added to group
	- Add multiple users to the domain
	- Trusting a user adds them to the group (can edit ~/www)
